  • 16 août 2021 - 16 octobre 2021
    Stage chez Vert Marine
    Durant ce stage dans le cadre de ma formation à l'AFPA, j'ai mis en place une application analytique du nom de VMA (Vert Marine Analytics). Le but de ce stage était de mettre en place une application qui générait des graphiques aux diverses statistiques selon le site VERT MARINE.
    J'ai pour cela crée une application avec les technologies suivantes :
    - Symfony 5 a été choisi pour ses bundles, sa modularité, son appel à la base donnée avec Doctrine et son modèle MVC facile à mettre en place.
    - Chart.Js a été choisi pour générer des graphiques professionnels et épurés.
    - MySQL a été choisi en tant que SGBD.
    - L'application utilise les langages basiques comme Javascript, PHP 7.4 et les langages de balisage HTML5 et CSS3.
